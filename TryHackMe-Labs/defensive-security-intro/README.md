# Objective

This room introduces the fundamental concepts of **defensive cybersecurity**.

Unlike offensive security, which focuses on exploiting vulnerabilities, defensive security focuses on **detecting, monitoring, and responding to cyber threats**.

The objective of this lab is to understand how organizations protect their infrastructure and investigate suspicious activity.

---

# Task 1 — Introduction to Defensive Security

This section explains the concept of defensive security and its importance in protecting digital environments.

Defensive security focuses on identifying threats, monitoring systems, and preventing malicious activities before they cause damage.

Key concepts introduced:

- Threat detection
- Security monitoring
- Incident response
- Infrastructure protection

---

# Task 2 — Responsibilities of Defensive Security

This task introduces the daily responsibilities of cybersecurity professionals working in defensive roles.

Typical responsibilities include:

- Monitoring system logs
- Investigating suspicious activities
- Analyzing network traffic
- Detecting potential intrusions
- Responding to security incidents

These activities are commonly performed by professionals working in **Security Operations Centers (SOC)**.

---

# Task 3 — Defensive Security in Practice

In real-world environments, security teams rely on multiple technologies to monitor systems and detect threats.

Some common defensive security tools include:

- SIEM (Security Information and Event Management)
- Intrusion Detection Systems (IDS)
- Log analysis tools
- Network monitoring tools

These technologies help analysts identify unusual behavior and investigate potential cyber attacks.
