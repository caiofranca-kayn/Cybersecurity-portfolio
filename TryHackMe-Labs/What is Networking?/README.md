## 🎯 Objective

---

## 📝 Task 1 — What is Networking?

### 🔑 Key Concept
Networking allows devices to **communicate**, exchange data, and access shared resources.

### 📚 Main Learnings
- A network is a collection of devices connected to share resources  
- Networking is essential for communication between computers, servers, and other devices  
- Understanding networking basics is critical for troubleshooting and cybersecurity  

---

## 🌐 Task 2 — What is the Internet?

### 🔑 Key Concept
The Internet is a **network of networks**, connecting millions of devices worldwide.

### 📚 Main Learnings
- Devices use **IP addresses** to communicate globally  
- Understanding the Internet structure helps track how data travels between devices  
- Knowledge of Internet architecture is crucial for cybersecurity  

---

## 🖥 Task 3 — Identifying Devices on a Network

### 🔑 Key Concept
Every networked device has an **IP address** and sometimes a **MAC address**, which uniquely identifies it.

### 📚 Main Learnings
- Devices include computers, servers, routers, switches, and IoT devices  
- IP addresses allow devices to communicate within a network or across the Internet  
- Identifying devices is key for network management and security  

---

## 📡 Task 4 — Ping (ICMP)

### 🔑 Key Concept
Ping tests **connectivity** between devices and measures network **latency** using ICMP.

### 📚 Main Learnings
- Ping uses ICMP packets to check if a device is reachable  
- Responses indicate network availability and delay  
- Understanding Ping is essential for diagnosing network issues and monitoring connectivity  

### 🛠 Tools Introduced
- Ping (ICMP)  
- Network troubleshooting basics
