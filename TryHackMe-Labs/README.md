## Objective

This room introduces the fundamentals of **defensive cybersecurity**, focusing on how organizations detect, analyze, and respond to cyber threats.

The lab demonstrates how security teams monitor systems, identify malicious activity, and protect infrastructure from attacks.

---

## Task 1 — Introduction to Defensive Security

The first task explains what defensive security is and how it differs from offensive security.

While offensive security focuses on finding vulnerabilities, defensive security focuses on **protecting systems and detecting attacks**.

Key concepts introduced:

- Monitoring systems and networks
- Detecting suspicious activity
- Protecting digital infrastructure
- Responding to security incidents

---

## Task 2 — Responsibilities of Defensive Security

This section presents the main responsibilities of defensive security professionals.

Some of the core activities include:

- Monitoring network traffic
- Investigating suspicious events
- Analyzing security logs
- Detecting potential intrusions
- Responding to security incidents

These activities are commonly performed by professionals working in **Security Operations Centers (SOC)**.

---

## Task 3 — Defensive Security in Practice

This task demonstrates how defensive security works in real-world environments.

Security teams rely on various tools and technologies to detect threats, including:

- Intrusion Detection Systems (IDS)
- Security Information and Event Management (SIEM)
- Log analysis tools
- Network monitoring systems

These technologies help security analysts identify malicious behavior and investigate potential cyber attacks.
