## 🎯 Objective

The **Starting Point** track introduces the fundamental concepts of **penetration testing and offensive cybersecurity**.

This guided learning path focuses on the essential building blocks required for beginners entering the cybersecurity field, including connecting to labs, performing enumeration, and interacting with network services.

---

## 🧠 Skills Developed

During this learning path, the following foundational skills were practiced:

- Connecting to VPN environments for penetration testing labs
- Basic **network enumeration**
- Identifying open ports and services
- Interacting with common network services
- Understanding the basics of remote system access

---

## 🚀 Conclusion

Completing the **Starting Point Tier 0** machines helped reinforce the fundamental steps used in penetration testing:

1. Reconnaissance  
2. Enumeration  
3. Service interaction  
4. Exploitation of simple misconfigurations  

These labs serve as a **solid introduction to offensive cybersecurity and penetration testing workflows**.
